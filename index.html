Project Proposal: Easy Math

Team Members

  • Brandon Gutierrez

Application Overview

This calculator is going to perform basic math such as addition, subtraction, multiplication, and division. It is aimed at providing a convenient and straightforward tool for users to perform everyday calculations with ease.

Technology Overview

  • Languages: Phyton
  • Libraries: None
  • Platforms: MAC and Windows
  • Hosting: GitHub

Proposal Last Updated

11/4/2023

Feature List

  1. Add 2 numbers
  2. Subtract 1 number by another
  3. Multiply 2 numbers
  4. Divide 1 number by another

Future Feature List

  • Scientific Calculator Mode
